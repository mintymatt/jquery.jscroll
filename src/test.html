Usage:

This is a script that provides scrolling on element click. Note scrolling is considered from the location of the element relative to the window.

<a href="#" class="jscroll" data-jscroll-vh="1" data-jscroll-offset="8">Scroll 100vh Down</a>


Init:

define link for scrolling by setting class name "jscroll".


data-jscroll-vh=(int)

Provide a (v)ertical (h)eight multplier. If none or 0 set, the page will scroll in the chosen direction until it hits an end. I.E data-jscroll=2 == 2*100vh


data-jscroll-offset = (int = pixel value)

apply an extra (pixel) offset to the scrolling. Useful if margins or paddings affect 100vh scrolling.


data-jscroll-up

By default scrolling is down. If you wish to scroll up instead, set this data attribute (no value needed)
